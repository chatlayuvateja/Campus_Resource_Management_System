==================================================================
  CRMS — Campus Resource Management System
  FIXED VERSION — HOW TO RUN
==================================================================

STEP 1: SET UP DATABASE
-----------------------
1. Open MySQL Workbench
2. Click File > Open SQL Script
3. Choose "fix_database.sql" from this folder
4. Click the lightning bolt (Execute All)
5. You should see: "Database ready — all tables created/fixed, resources seeded."

This creates all tables AND seeds them with 40 campus resources
(classrooms, labs, sports, library, equipment).

STEP 2: CONFIGURE DATABASE PASSWORD
------------------------------------
Open:  src/com/crms/db/DBConnection.java
Find:  conn = DriverManager.getConnection(..., "root", "yuva@2005");
Change "yuva@2005" to YOUR MySQL root password.

STEP 3: COMPILE
---------------
1. Open compile.bat
2. Change TOMCAT_HOME to your actual Tomcat folder path
3. Double-click compile.bat
4. You should see: "SUCCESS!"

STEP 4: DEPLOY TO TOMCAT
-------------------------
Copy this entire "CRMS_final" folder to:
  C:\apache-tomcat-9.0.xx\webapps\

Start Tomcat (bin\startup.bat)

Open browser:  http://localhost:8080/CRMS_final/login.html

STEP 5: CREATE AN ADMIN USER
-----------------------------
Run this SQL in MySQL Workbench:
  INSERT INTO campus_rms.users
    (username, password, full_name, email, id_number, role)
  VALUES
    ('admin', 'admin123', 'Administrator', 'admin@campus.edu', 'ADM001', 'admin');

Then login with:  admin / admin123
